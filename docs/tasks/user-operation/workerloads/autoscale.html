弹性伸缩​

kubegems 弹性扩容基于kubernetes Horizontal Pod Autoscaler,只是将其在界面上简化抽象成表单，基于cpu和内存进行扩缩容;

目前只能针对workloads(Deployment StatefulSet)进行设置

字段说明
最大cpu使用率限制pods的CPU平均使用率，超过这个值将触发扩容
最大内存使用率限制pods的内存平均使用率，超过这个值将触发扩容
最小副本数缩容最小副本数下限
最大副本数扩容最大副本数上限
